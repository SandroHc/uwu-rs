# uwu_cli

A uwu CLI tool inspired by [Daniel Liu's uwu](https://github.com/Daniel-Liu-c0deb0t/uwu).

## Quick start

Install the executable with:

```shell
cargo install uwu_cli
```

Then use it:

```shell
# directly from the arguments
uwu Hello world!

# from a file
uwu --file uwu.txt

# from stdin
uwu <<EOF
Life in uwu land,
Is fantastic!
EOF
```
